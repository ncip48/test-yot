## Installation Soal 1

- Clone the Repo
- Install all library needed to environment by using command

```bash
$ cd soal1
$ npm install
```

## Usage

- Start server with command:

```bash
$ cd soal1
$ npm start
```

Then open [http://localhost:3000/](http://localhost:3000/)
